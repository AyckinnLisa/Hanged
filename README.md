### *USAGE*
* Download [Python](https://www.python.org/downloads/).
* Open the terminal for Unix or CMD for Windows
* Go to the game folder : ```cd /game_folder_path```
* Run the game : ```python3 hanged.py```
* To quit the game, enter : ```!q```
